## Rocket编程指南

欢迎来到Rocket！

这是Rocket v0.4的官方指南。它旨在用作使用Rocket和Rust编写Web应用程序的起点。该指南还旨在为经验丰富的Rocket开发人员提供参考。本指南采用对话形式。有关带有示例的纯技术文档，请参阅[API文档](https://api.rocket.rs/v0.4)。

该指南分为几个部分，每个部分重点介绍Rocket的不同方面。这些部分是：

- **[简介](/rocketdoc/Introduction/Audience.html)：**介绍Rocket及其哲学。
- **[快速开始](/rocketdoc/Quickstart/Running-Examples.html)：**介绍运行第一个Rocket应用程序所需的最少步骤。
- **[入门](/rocketdoc/Getting-Started/Installing-Rust.html)：**简要介绍如何运行您的第一个Rocket应用程序。
- **[概述](/rocketdoc/Overview/Lifecycle.html)：**描述了Rocket的核心概念。
- **[请求 Requests](/rocketdoc/Requests/Methods.html)：**讨论处理请求：控制流，解析和验证。
- **[响应 Responses](/rocketdoc/Responses/Responder.html)：**讨论生成响应。
- **[托管状态 State](/rocketdoc/State/Managed-State.html)：**如何在Rocket应用程序中管理状态。
- **[整流罩 Fairings](/rocketdoc/Fairings/Overview.html)：**概述了Rocket的结构化中间件。
- **[测试](/rocketdoc/Testing/Local-Dispatching.html)：**如何对Rocket应用程序进行单元和集成测试。
- **[配置](/rocketdoc/Configuration/Environment.html)：**如何配置Rocket应用程序。
- **[Pastebin 教程](/rocketdoc/Pastebin/Finished-Product.html)：**关于如何使用Rocket创建**[ pastebin](/rocketdoc/Pastebin/Finished-Product.html)**的教程。
- **[总结](/rocketdoc/Conclusion/Getting-Help.html)：**总结了指南并讨论了学习的后续步骤。

## 获得帮助

官方的社区支持频道是[Mozilla IRC 服务](https://wiki.mozilla.org/IRC)上的 `#rocket` IRC频道，地址在`irc.mozilla.org`和在Matrix上桥接的 [Rocket room](https://riot.im/app/#/room/#mozilla_#rocket:matrix.org)。如果您不熟悉IRC，我们建议您通过 [Matrix via Riot](https://riot.im/app/#/room/#mozilla_#rocket:matrix.org) 或通过 [Kiwi web IRC client](https://kiwiirc.com/client/irc.mozilla.org/#rocket) 聊天。您可以通过Mozilla的IRC入门指南了解更多关于IRC的信息。
