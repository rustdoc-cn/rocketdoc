总结

我们希望您同意，Rocket是对Web框架的一种崭新的尝试。与任何软件项目一样，Rocket仍然存在。总有需要改进的地方，我们很乐意采纳最好的想法。如果您有任何想法，请提交问题。

获取帮助§

官方的社区支持频道是Mozilla IRC 服务上的 #rocket IRC频道，地址在irc.mozilla.org和在Matrix上桥接的 Rocket room。如果您不熟悉IRC，我们建议您通过 Matrix via Riot 或通过 Kiwi web IRC client 聊天。您可以通过Mozilla的IRC入门指南了解更多关于IRC的信息。
